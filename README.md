# Aprendizajes del módulo Fundamentos del Desarrollo Web
 **Proyecto 4 Introducción a Flexbox **

 Se siguio trabajando con la estructura del html entregado anteriormente, unicamente se elimino parte de codigo incesesario y se añadieron partes fundamentales para su estructura, el tema siguio siendo el mismo (portafolio), unicamente

En este proyecto se implementó: 
  * Maquetación de secciones usando **Flexbox** como la barra de navegación (`navbar`) 
  * Sección hero en la cual se usa de presentacion en el portafolio 
  * Uso de **Google Fonts** para mejorar la tipografía. 
  * Uso de una paleta de colores llamada `Monochrome Beach`.
- Uso de atributos de *flexbox* para el acomodo de secciones como:
  - `display: flex` para activar el modelo Flexbox en contenedores. 
  - `flex-direction: row | column` para organizar elementos en fila o columna. 
  - `justify-content: space-around | space-between | center | flex-end` para distribuir elementos en el eje principal. 
  - `align-items: center | flex-start | flex-end` para alinear elementos en el eje cruzado. 
  - `flex: 1` para que las cards crezcan y ocupen espacio proporcional. 
  - `align-content` para controlar la distribución cuando hay varias filas.
